# RandomCreations

[Hugo](https://gohugo.io)

## Local dev

- **Node**: install `fnm` ([Schniz/fnm](https://github.com/Schniz/fnm)) and run `fnm use` (this repo uses `.node-version`).
- **Package manager**: `pnpm` via Corepack.

```sh
brew install fnm
fnm use
corepack enable
pnpm install
hugo serve
```
